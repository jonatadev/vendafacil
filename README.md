# VendaFácil

Uma aplicação de e-commerce moderna desenvolvida com React, TypeScript e Material-UI.

## Funcionalidades

- Catálogo de produtos
- Carrinho de compras com persistência local
- Interface responsiva
- Cálculo automático de frete
- Controle de estoque

## Tecnologias

- React 18
- TypeScript
- Material-UI (MUI)
- Vite
- Context API

## Como executar

```bash
npm install
npm run dev
```
